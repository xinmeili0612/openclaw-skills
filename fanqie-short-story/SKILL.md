# 番茄小说短篇创作助手

## 功能特点
- 📖 **故事生成**：根据用户提示生成短篇故事
- 📝 **情节发展**：扩展故事情节，丰富内容
- 👤 **人物设定**：创建生动的人物角色
- 🎭 **风格转换**：支持不同风格的小说创作（玄幻、科幻、言情等）

## 使用方法

### 生成故事
```bash
openclaw skill fanqie-short-story story "在一个神秘的小镇上，发生了一系列离奇的事件"
```

### 扩展情节
```bash
openclaw skill fanqie-short-story plot "主角在森林中发现了一个古老的遗迹"
```

### 创建人物
```bash
openclaw skill fanqie-short-story character "一个神秘的流浪歌手"
```

### 风格转换
```bash
openclaw skill fanqie-short-story style "悬疑" "在一个雨夜，侦探接到了一个神秘的电话"
```

## 配置说明
- 支持自定义故事长度和风格
- 可以保存生成的故事到文件
- 支持续写已有的故事内容
