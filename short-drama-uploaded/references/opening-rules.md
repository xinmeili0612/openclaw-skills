# 开篇黄金5秒法则

## 核心原则
短剧观众的注意力极其宝贵。
前5秒没有钩子，观众直接划走。
前30秒没有冲突，观众流失率超过60%。

## 第一集开篇必须做到
1. 前5秒：建立强烈视觉或情感冲击
2. 前30秒：明确主角处境 + 引出核心冲突
3. 第一场戏结束前：观众必须产生"然后呢？"的疑问

## 6种经典开场模板

### 1. 直接打脸型
开场即是主角被羞辱/欺负的高光时刻，然后迅速反转或埋下复仇伏笔。
示例：婚礼现场，新郎当众宣布换人，女主被推下台——但女主眼神变得冰冷，嘴角微微上扬。
适用题材：重生穿越、霸总、逆袭

### 2. 身份反转型
开场展示主角"弱"的一面，第一集末尾或第二集初揭示真实身份。
示例：外卖小哥被保安拦在豪华写字楼门口，镜头切到他手机上密密麻麻的未接来电——全是"董事长"。
适用题材：战神归来、霸总、都市情感

### 3. 极端困境型
开场即是主角面临人生最低谷，逼迫观众产生同情和"他/她怎么办"的悬念。
示例：医院走廊，主角跪在地上求医生救她的孩子，口袋里只剩几十块钱。
适用题材：家庭伦理、励志逆袭、萌宝

### 4. 倒叙悬念型
开场是故事高潮片段（打脸/爽点），然后字幕"三个月前"，开始讲述来龙去脉。
示例：开场：主角站在公司顶楼，俯视跪在地上的反派，说"当初是谁说我一无是处？"——三个月前……
适用题材：所有题材通用，尤其适合悬疑、逆袭

### 5. 高甜钩子型
开场即是男女主的高甜互动，迅速建立CP感，让观众"磕到了"。
示例：清晨，男主替熟睡的女主挡住阳光，低头看着她，旁白："我以为我不会再爱了。"
适用题材：甜宠、霸总、都市情感

### 6. 重生起始型
开场展示"上一世"的惨烈结局（被背叛/死亡），然后主角重生，眼神从茫然变为坚定。
示例：主角被推下悬崖，睁眼发现回到了五年前，嘴角慢慢浮现出一个冷笑。
适用题材：重生穿越、复仇类
