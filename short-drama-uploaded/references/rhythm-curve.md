# 节奏曲线设计

## 单集三幕微结构

每一集都是一个完整的小故事，内部有自己的节奏：

```
前30秒（钩子）→ 中间1分钟（冲突升级）→ 最后30秒（爽点释放+下集钩子）
```

- **前30秒**：必须有一个让观众无法划走的钩子（悬念/冲突/甜点/笑点）
- **中间部分**：冲突持续升级，不能平铺直叙，每隔30秒要有一个小高潮
- **最后30秒**：当集爽点释放，同时埋下下集钩子，让观众立刻点下一集

## 全剧四阶段节奏波形

```
阶段一（建置期，约前15%集数）：
波形：低→中，逐步上升
任务：建立世界观、人物登场、初始冲突埋下

阶段二（升温期，约15%-50%集数）：
波形：中→高，多个小高潮，有起伏
任务：冲突持续升级，多次打脸，感情线推进，付费卡点1-2

阶段三（高潮期，约50%-85%集数）：
波形：高→极高，持续高强度
任务：重大反转，身份揭露，终极对决前的积累，付费卡点3-4

阶段四（收尾期，约85%-100%集数）：
波形：极高→释放
任务：终极打脸，情感爆发，大结局，所有伏笔收回
```

## 节奏禁忌
- 连续3集以上没有爽点 → 观众流失
- 连续2集以上都是铺垫对话 → 节奏拖沓
- 高潮后没有给观众喘息的甜点场景 → 情绪疲劳
- 结局集数过多（超过5集）→ 拖尾感
