# AI小说创作助手

## 功能特点
- 📖 **情节生成**：根据用户提示生成小说情节发展
- 👤 **人物设定**：创建丰富的人物角色和背景故事
- 📝 **章节写作**：生成小说章节内容
- 🎭 **风格转换**：支持不同风格的小说创作（玄幻、科幻、言情等）

## 使用方法

### 生成情节
```bash
openclaw skill novel-writer plot "在未来世界，一个普通少年意外获得了超能力"
```

### 创建人物
```bash
openclaw skill novel-writer character "一个神秘的女杀手，拥有双重身份"
```

### 写章节
```bash
openclaw skill novel-writer chapter "第一章：神秘的遗迹"
```

### 风格转换
```bash
openclaw skill novel-writer style "科幻" "在一个遥远的星球上，发生了一场离奇的事件"
```

## 配置说明
- 支持自定义小说风格和字数
- 可以保存生成的内容到文件
- 支持续写已有的小说内容
