# 短剧剧本生成技能

基于微信公众号文章《短剧动态漫创作，最重要的就是剧本！剧本创作Skill 发布！》开发的短剧剧本生成工具。

## 🎯 核心功能

### 1. 剧本定位分析
- 市场趋势分析
- 目标受众定位
- 竞争环境评估
- 创作方向建议

### 2. 角色档案生成
- 主角/配角背景故事
- 角色性格特征设计
- 人物关系网络构建
- 冲突矛盾设置

### 3. 剧本大纲创作
- 三幕结构设计（开端-发展-高潮-结局）
- 情节转折点安排
- 爽点节奏控制
- 商业化元素融入

### 4. 标准格式输出
- 文学剧本标准格式
- 场景头（内/外景、地点、时间）
- 景别提示（全景、中景、近景、特写）
- 动作描写和语气指示
- 音乐和音效提示

## 📝 使用场景

### 60集短剧创作
- 每集1-3分钟剧本
- 连续剧情设计
- 商业化变现结构

### 动态漫剧本
- 视觉化场景描述
- 动画分镜指导
- 音效音乐配合

### 出海创作
- 跨文化剧本改编
- 国外市场偏好分析
- 本地化内容创作

## 💰 市场参考

### 收入标准
- **新人作者**：2万/60集，15-20天完成，月收入2-6万
- **资深作者**：月收入5-10万，分成10-30万，年收入可达100万
- **投稿要求**：原创剧本，格式正确，有完整大纲

### 市场机会
- **国内市场**：竞争激烈但需求大，平台收稿要求高
- **国外市场**：编剧资源稀缺，蓝海市场机会多
- **出海优势**：国内成熟经验+国外市场需求=竞争优势

## 🚀 工作流程

### 完整创作流程
```
1. 市场分析 → 确定创作方向
2. 角色设计 → 创建人物档案
3. 大纲创作 → 设计故事结构
4. 分集创作 → 编写详细剧本
5. 格式规范 → 输出标准格式
6. 投稿准备 → 准备投稿材料
```

- **ComfyUI/N8N**：工作流自动化
- **AI生图工具**：场景视觉化

## 📋 使用示例

### 生成剧本大纲
```bash
# 生成爱情短剧大纲
short-drama-script generate-outline \
  --genre "爱情" \
  --episodes 60 \
  --theme "霸道总裁爱上我" \
  --target-audience "女性18-35岁"

# 直接显示大纲内容（不保存文件）
short-drama-script generate-outline \
  --genre "穿越" \
  --episodes 60 \
  --theme "现代女医生穿越古代" \
  --show

# 生成穿越短剧大纲
short-drama-script generate-outline \
  --genre "穿越" \
  --episodes 60 \
  --theme "现代女医生穿越古代" \
  --commercial-elements "宫斗、医术、爱情"
```

### 创建角色档案
```bash
# 创建主角角色档案
short-drama-script create-character \
  --name "林婉儿" \
  --role "女主角" \
  --age "25" \
  --personality "聪明独立、善良坚韧" \
  --background "现代外科医生，意外穿越到古代"

# 创建反派角色档案
short-drama-script create-character \
  --name "慕容雪" \
  --role "反派女配" \
  --age "22" \
  --personality "嫉妒心强、工于心计" \
  --motivation "争夺太子妃位置"
```

### 生成标准格式剧本
```bash
# 将大纲转为标准剧本格式
short-drama-script format-script \
  --input "大纲.txt" \
  --output "剧本.docx" \
  --format "standard"

# 直接显示剧本内容（不保存文件）
short-drama-script format-script \
  --input "大纲.txt" \
  --show

# 批量处理多集剧本
short-drama-script batch-format \
  --input-dir "分集大纲/" \
  --output-dir "标准剧本/" \
  --episodes 60
```

### 快速剧本生成（直接显示）
```bash
# 快速生成并显示剧本
short-drama-script quick-generate \
  --theme "真千金是学霸" \
  --episode 1

# 列出可用主题
short-drama-script quick-generate --list

# 生成第2集
short-drama-script quick-generate \
  --theme "真千金是学霸" \
  --episode 2
```

## 🔧 技术实现

### 剧本格式规范
```python
# 标准剧本格式示例
场景1：内景 医院手术室 日
【全景】手术室内，无影灯下，林婉儿正在紧张地进行手术。
【近景】林婉儿额头渗出细密汗珠，眼神专注。
【特写】手术刀精准地切割。
林婉儿（冷静）：止血钳。
护士（紧张）：医生，病人血压下降！
【音乐】紧张急促的背景音乐起。
```

### 爽点节奏控制
- **第1-5集**：建立主角困境，引发观众共鸣
- **第6-15集**：小高潮不断，制造期待感
- **第16-30集**：主要冲突爆发，情节转折
- **第31-45集**：情感升华，关系发展
- **第46-55集**：最终冲突，高潮部分
- **第56-60集**：结局收尾，留下续集悬念
